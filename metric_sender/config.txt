# input format

# <metric-type> <metric-count> <telegraph-host:port> <total-iteration-count> <sleep-between-iteration-in-seconds>
# <metric-type> can be one of [apache | iis_server | oracle | tomcat | sql_server]
# Minimun sleep-between-iteration is set to 60 seconds
#
# Example:
# send 5 count of apache metrics to telegraf server running in host 127.0.0.1:8092 for 10 times with 60 seconds interval
# apache 5 127.0.0.1:8092 10 60

apache 5 127.0.0.1:8092 10 60
iis_server 5 127.0.0.1:8093 10 60
oracle 5 127.0.0.1:8094 10 60
tomcat 5 127.0.0.1:8095 10 60
sql_server 5 127.0.0.1:8096 0 60
